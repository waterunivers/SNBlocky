Drop your modpack files into this folder.

Supported formats: .zip, .mrpack, .jar, or any file type.

Each file will automatically appear as a download link in the Discord bot's
📦 Downloads button response — no code changes needed.

Examples:
  CrossroadsSMP-Modpack-v1.0.mrpack
  ResourcePack-Community.zip
  OptionalPvPPack.zip

Files are served via the bot's built-in Express web server at:
  http://YOUR_SERVER:PORT/downloads/filename
